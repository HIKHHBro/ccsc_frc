    - CAN ID: 5~6

3. 自动收缩
### 抓取 
1. PCM
    - CAN ID :7
    - PCM channel: 0,1
2. 电机
    - Neo+SPA
    - PWM: 0
#### 功能
1. 气孔控制
2. 抓球电机pwm控制
#### 执行动作
1. 气阀
    1. 放下  - 1 true - 2 true
    2. 抓取  - 1 false - 2 false
    3. 抬起  - 1 true - 2 false
2. 电机 顺时针 速度为负值

### 发射
1. 1. 电机
    - 型号: Neo
    - PWM ID: 1~4
#### 功能
1. 摄像头识别
2. 云台电机位置控制
3. 传送弹丸电机pwm控制
#### 执行动作
1. 传送1 电机 顺时针
2. 传送2 电机 逆时针
3. 摩擦轮 1和2都是逆时针
4. 云台pitch轴  顺时针
### 转盘
1. 旋转控制
2. 指定颜色位置控制
## 底盘
